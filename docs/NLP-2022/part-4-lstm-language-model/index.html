Part 4 - LSTM Language Model

In this notebook, we will train a LSTM model to generate text.

Such models fall under the umbrella of Language Models. There are two main categories of language models - masked language models(MLM) and causal language models (CLM). We will cover this topic in more detail under the Transformers section. For now we will just train a CLM that learns to generate text by predicting one token at a time.

We will be training a word/token level LSTM but it can be trained at character level as well(vocabulary size will be much smaller at character level).

Within each forward pass, the model will receive indices for a sequence and will be tasked to predict the the index for next token/word at each time step. For example:

Input : The boy who lived

Output : boy who lived .

class TextProcessor:
    def __init__(self, urls: List[str]):
        self.urls = urls

    def fetch_text(self):
        self.text = ""
        for i, url in enumerate(self.urls):
            r = requests.get(url)
            self.text += "\n" + r.text.strip().lower()
            print(f"Fetched {i+1}/{len(self.urls)} urls ...")

    def process_text(self):
        print("Processing text ...")
        self.text = re.sub("\n{2,}", "\n", self.text)
        self.text = " ".join(re.findall("[\w\n]+", self.text))
        self.text = self.text.replace("\n", "").strip()
        self.text = re.sub("page \d{1,3} harry potter and the philosophers stone j k rowling ",
                           "", self.text)
        self.tokens = self.text.split(" ")
        print(f"{len(self.tokens)} tokens")
        self.token_counter = Counter(self.tokens)
        self.unique_tokens = sorted(self.token_counter, key=self.token_counter.get, reverse=True)
        print(f"{len(self.token_counter)} unique tokens")

    def prepare_vocab(self):
        self.i2w = {ix: word for ix, word in enumerate(self.unique_tokens)}
        self.w2i = {word: ix for ix, word in enumerate(self.unique_tokens)}
        print(f"Vocab size: {len(self.w2i)}")
        self.words_indexes = [self.w2i[w] for w in self.tokens]

Preparing the torch Dataset and Dataloader​

The Dataloader will enable us to send data in batches during the forward pass of the model training.

class Dataset(torch.utils.data.Dataset):
    def __init__(self, text_processor: TextProcessor, seq_len: int=100, ):
        self.seq_len = seq_len
        self.text_processor = text_processor

    def __len__(self):
        return len(self.text_processor.words_indexes) - self.seq_len

    def __getitem__(self, index):
        return (
            torch.tensor(self.text_processor.words_indexes[index: index+self.seq_len]).to(device),
            torch.tensor(self.text_processor.words_indexes[index+1: index+self.seq_len+1]).to(device),
        )
dataset = Dataset(seq_len=100, text_processor=text_processor)

Here we are creating the Dataset with fixed sequence lenghts. A better approach would be to keep the sequence length dynamic with a max_seq_len param to pad shorter sequence lenghts.

Pytorch also provides a feature to pack variable length sequences together using torch.nn.utils.rnn.pack_padded_sequence. Here we are using fixed length sequences for simplicity.

Creating the LSTM model using torch.nn.Module​

class Model(nn.Module):
    def __init__(self, dataset):
        super(Model, self).__init__()
        self.lstm_size = 128
        self.embedding_dim = 128
        self.num_layers = 2
        self.seq_len = dataset.seq_len
        self.n_vocab = len(dataset.text_processor.w2i)

        self.embedding = nn.Embedding(
            num_embeddings=self.n_vocab,
            embedding_dim=self.embedding_dim,
        )
        self.lstm = nn.LSTM(
            input_size=self.embedding_dim,
            hidden_size=self.lstm_size,
            num_layers=self.num_layers,
            dropout=0.2,
        )
        self.linear = nn.Linear(self.lstm_size, self.n_vocab)

    def forward(self, x, prev_state):
        embed = self.embedding(x)
        output, state = self.lstm(embed, prev_state)
        logits = self.linear(output)
        return logits, state

    def init_state(self, sequence_length=None):
        seq_len = sequence_length if sequence_length else self.seq_len
        return (torch.zeros(self.num_layers, seq_len, self.lstm_size).to(device),
                torch.zeros(self.num_layers, seq_len, self.lstm_size).to(device))
model = Model(dataset)
model.to(device)

for epoch in range(N_EPOCHS):
    state_h, state_c = model.init_state(100)

    for i, (x, y) in enumerate(dataloader):
        optimizer.zero_grad()

        loss = criterion(y_pred.transpose(1, 2), y)

Generating text​

def predict(dataset, model: Model, text: str, next_words: int=100):
    model.eval()

    words = text.split(' ')
    state_h, state_c = model.init_state(len(words))

    for i in range(next_words):
        x = torch.tensor([[dataset.text_processor.w2i[w] for w in words[i:]]]).to(device)
        y_pred, (state_h, state_c) = model(x, (state_h, state_c))

        last_word_logits = y_pred[0][-1]
        p = torch.nn.functional.softmax(last_word_logits, dim=0).detach().cpu().numpy()
        word_index = np.random.choice(len(last_word_logits), p=p)
        words.append(dataset.text_processor.i2w[word_index])

    return words
